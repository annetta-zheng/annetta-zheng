   ```
      * B III II I maa (az) (mn)
      asb 
   ```
    
   Output: 
    ```
      wtt
    ```

```
    abcdefghijklmnopqrstuvwxyz

             (gt) (hs) (ir) (jq) (kp) (lo) (mn)
```
